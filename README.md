# Life Cycle Transformation Explorer

## 📌 Overview

This is a simple web-based simulation that demonstrates direct and indirect life cycles in organisms.
It shows how organisms grow and develop through different stages.
The concept is explained in a clear and visual way.

## 🚀 Features

* Demonstrates life cycle stages
* Simple and interactive design
* Easy to understand concept
